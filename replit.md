# iSynera AI Healthcare Platform

## Overview

iSynera AI Healthcare Platform is a comprehensive HIPAA-compliant healthcare automation system designed to streamline medical workflow management through intelligent, adaptive AI technology. The platform focuses on automating patient intake, referral processing, eligibility verification, homebound assessments, and billing operations while maintaining strict healthcare compliance standards.

## System Architecture

### Frontend Architecture
- **Framework**: React 18 with TypeScript
- **Build Tool**: Vite for fast development and optimized production builds
- **UI Components**: Shadcn/ui component library with Radix UI primitives
- **Styling**: Tailwind CSS with custom healthcare theme variables
- **State Management**: TanStack Query (React Query) for server state management
- **Routing**: Wouter for lightweight client-side routing
- **Authentication**: JWT-based authentication with session management

### Backend Architecture
- **Runtime**: Node.js with TypeScript
- **Framework**: Express.js with session-based authentication
- **Database**: PostgreSQL with Drizzle ORM for type-safe database operations
- **Database Provider**: Neon serverless PostgreSQL
- **API Design**: RESTful APIs with comprehensive error handling
- **File Processing**: Multer for file uploads and document processing

### AI/ML Integration
- **Primary AI Provider**: OpenAI GPT-4.1 for intelligent document processing and analysis
- **Secondary AI Provider**: Anthropic Claude for specialized healthcare tasks
- **Voice Processing**: Integration with speech-to-text and text-to-speech capabilities
- **Document Analysis**: OCR processing with intelligent data extraction
- **Healthcare Intelligence**: Specialized AI agents for referral analysis, eligibility verification, and homebound assessments
